## Dev
http://localhost:9000/

### Project setup/install
```
yarn
```

### Compiles and hot-reloads for development
```
yarn start
```

### Compiles and minifies for production
```
yarn build
```

### Update packages
```
yarn update
```

### ESlint --fix
```
yarn lint
```

### Preview public build
```
serve public
```
